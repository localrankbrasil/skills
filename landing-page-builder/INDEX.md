## ⚙️ The Driver (driver.mjs)

CLI tool for generating landing pages locally:

```bash
# List available styles
node driver.mjs --list-styles

# Generate preview files for specific aesthetics
node driver.mjs --preview minimalist maximalist brutalist

# Generate a full landing page
node driver.mjs --generate minimalist my-page.html
```

**Note:** Most users won't need this — just ask Claude to create your page!

## ❓ FAQ

**Q: Can I generate multiple landing page variants?**
A: Yes! Ask Claude to create 3 versions with different aesthetics or copy. A/B test them.

**Q: Can I export as Next.js instead of HTML?**
A: Yes! Ask Claude to generate a Next.js project instead. It'll include TypeScript, Tailwind, ShadCN components.

**Q: How do I connect forms to email capture?**
A: HTML version has placeholder form. Connect to Mailchimp, ConvertKit, or Zapier. See README.md for details.

**Q: Can I host this on my own domain?**
A: Yes! Deploy anywhere. GitHub Pages, your server, VPS, etc.

**Q: How do I track conversions?**
A: Add Google Analytics or Hotjar to the HTML. See README.md for setup code.

**Q: Can I modify the code after generation?**
A: Yes! All CSS and HTML are editable. Change colors, fonts, text, layout — anything.

**Q: Does this work for mobile?**
A: Yes! Mobile-first responsive design. Test on real devices (not just browser devtools).

**Q: How long does a landing page take to create?**
A: 15-30 minutes from brief to live page. Depends on customization needed.

## 📈 Success Metrics

After deploying, track:
- **Conversion Rate** — % of visitors who click CTA (target: 2-5%)
- **Click-Through Rate** — % who click main CTA button (target: 5-15%)
- **Scroll Depth** — How far down do people go? (target: 60%+)
- **Time on Page** — How long do they stay? (target: 1-3 min)
- **Traffic Source** — Where are visitors coming from?
- **Device Type** — Mobile vs desktop performance

Optimize based on data, not guesses!
