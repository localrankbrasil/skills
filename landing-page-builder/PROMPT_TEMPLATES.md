# Landing Page Builder — Prompt Templates

These are ready-to-use prompts for generating landing pages. Customize with your specific details.

---

## Template 1: SaaS Product (Default)

```
Create a high-converting landing page for [PRODUCT_NAME].

**Product Overview:**
- What it does: [1-sentence description]
- Target audience: [Who uses it? Job titles? Industries?]
- Main benefit: [Top 1 benefit]
- Key features: [3-5 key features]

**Design Preferences:**
- Aesthetic: [minimalist/maximalist/retrofuturistic/organic/editorial/brutalist]
- Tone: [professional/friendly/bold/playful]
- Primary color: [optional color preference]

**Content Preferences:**
- Include testimonials: [yes/no]
- Include pricing section: [yes/no]
- Include FAQ: [yes/no]
- Include comparison table: [yes/no]
- Call to action text: [e.g., "Start Free Trial", "Get Started"]

**Additional Notes:**
[Any specific requirements or preferences]

Generate the landing page as a single HTML file.
```

## Quick Customization Tips

### To Request Changes:
```
"Change the [section name] to emphasize [what]"
"Use [aesthetic] style instead"
"Replace [element] with [new element]"
"Make the CTA say [new text]"
"Add a section about [topic]"
```

### For Specific Styling:
```
"Use [color] as primary color"
"Change fonts to [font name]"
"Make the layout [wider/narrower]"
"Add [animation type] animation"
```

### For Content:
```
"Rewrite the headline to focus on [benefit]"
"Add [number] more testimonials"
"Include these features: [list]"
"Use this tone: [tone description]"
```

---

## Pro Tips

1. **Be Specific** — More details = better results
2. **Provide Examples** — Link to competitors or inspiration
3. **State Your Goal** — What conversion do you want? (signup/demo/purchase/download)
4. **Share Metrics** — What results do you want? (CTR, signups, etc.)
5. **Iterate** — Request changes and refine until perfect
