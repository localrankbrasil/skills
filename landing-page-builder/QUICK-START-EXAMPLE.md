## Example 1: SaaS Product (ProjectHub)

### Step 1: Ask Claude

```
Create a landing page for ProjectHub.

ProjectHub is a real-time project management tool for remote teams.
It solves the problem of scattered feedback and communication.

Target audience: Startups (10-50 people) and remote teams
Main benefit: Collaborate 3x faster with centralized feedback

Aesthetic: Minimalist (clean, professional, trustworthy)

Key features:
- Centralized feedback in one place
- Real-time collaboration
- Integrates with Slack and GitHub
- Works offline

Customer quote: "We reduced onboarding time by 80% with ProjectHub"

Call-to-action: "Start 14-Day Free Trial"
```

### Step 2: Claude Response

Claude generates complete HTML file with:
- Hero section with headline + CTA
- Social proof (customer logos)
- Problem statement
- Solution highlight
- 5 key benefits with icons
- Feature breakdown with screenshots
- Customer testimonials
- FAQ section
- Final CTA button

### Step 3: Customize (Optional)

```
Make the colors bolder with the Maximalist aesthetic.
Change the primary color to teal.
Add a pricing section showing three plans.
```

Claude updates the page instantly.

### Step 4: Deploy

**Copy the HTML** → **Drag to Netlify** → **Live in 30 seconds**

Result: Professional landing page ready to convert visitors ✅

## Templates & Prompts by Industry

### SaaS
```
Create a landing page for [SAAS_NAME].
It does: [1 sentence]
Target: [Who uses it]
Benefits: [Top 3 benefits]
Aesthetic: [Style choice]
```

### Service/Coaching
```
Create a landing page for my coaching business.
I help [Who] with [Problem] through [Solution].
Target audience: [Description]
Credentials: [Years/results]
Aesthetic: [Style choice]
Call-to-action: "Schedule Free Consultation"
```

### E-Commerce
```
Create a landing page for my product: [PRODUCT_NAME].
Price: [Cost]
Main benefits: [Top 3]
Target audience: [Who buys it]
Social proof: [Numbers/testimonials]
Aesthetic: [Style choice]
```

### Course/Digital Product
```
Create a landing page for my online course.
Course: [Title]
Topic: [What will students learn]
Target: [Who is it for]
Outcomes: [What will they achieve]
Aesthetic: [Style choice]
```

### Agency/Portfolio
```
Create a landing page for my creative agency.
Services: [List services]
Specialization: [What you're best at]
Target clients: [Who you want to work with]
Portfolio pieces: [Notable work]
Aesthetic: [Preferably maximalist/editorial for creative]
```

## Deployment Checklists

### Before Going Live

- [ ] All text reviewed and corrected
- [ ] Real company name used (not placeholder)
- [ ] Real images added (replace placeholders)
- [ ] Logo added
- [ ] Links work (test all buttons)
- [ ] Mobile tested (on real phone)
- [ ] Contact/form works
- [ ] Meta tags updated (title, description)
- [ ] Analytics code added (if desired)

### After Launch

- [ ] Monitor traffic
- [ ] Track conversion rate
- [ ] Check bounce rate
- [ ] Review user feedback
- [ ] A/B test headlines
- [ ] Optimize images
- [ ] Iterate based on data
