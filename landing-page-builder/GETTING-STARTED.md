# Landing Page Builder — Getting Started (5 minutes)

**You have 5 minutes? Do this:**

## The 5-Minute Setup

### 1. Open Claude (1 minute)
Use Claude Code, Claude web, or Claude in your IDE.

### 2. Ask for Your Landing Page (2 minutes)

Copy-paste this and fill in your details:

```
Create a landing page for [YOUR PRODUCT].

It does: [1 sentence description]
Target audience: [Who uses it?]
Main benefit: [Top 1 benefit]
Aesthetic: [Pick one: minimalist / maximalist / retrofuturistic / organic / editorial / brutalist]

Include these sections:
- [Section 1]
- [Section 2]
- [Section 3]

Call-to-action text: [e.g., "Start Free", "Get Started"]
```

### 3. Review the Output (1 minute)
Claude will generate HTML code for your landing page.

### 4. Copy & Deploy (1 minute)

**Option A: Netlify (Easiest)**
```
1. Go to netlify.com
2. Drag & drop the HTML file
3. Get a live URL instantly
```

**Option B: GitHub Pages**
```
1. Create GitHub repo
2. Upload HTML as index.html
3. Enable Pages in Settings
4. Live at: username.github.io/repo
```

**Option C: Vercel**
```
1. Go to vercel.com
2. Upload your HTML or folder
3. Get live URL
```

## Deployment Cheat Sheet

### Netlify
```
1. Visit https://netlify.com
2. Drag your HTML file
3. Done! (takes 30 seconds)
```

### GitHub Pages
```
git init
git add index.html
git commit -m "Landing page"
git push -u origin main
# Then enable Pages in repo settings
```

### Vercel
```
1. Visit https://vercel.com
2. Import project or drag file
3. Done! (auto-deploys)
```

### Heroku / Other Hosting
```
1. Upload HTML file via FTP/SFTP
2. Point domain to your host
3. Enable SSL (Let's Encrypt free)
```
